# 💰 Expense Tracker Pro

A full-stack Expense Tracker application built using the MERN Stack.

## 🚀 Live Demo

Frontend: https://self-tracker-xi.vercel.app

Backend API: https://self-tracker-1mv0.onrender.com

## ✨ Features

* User Registration & Login
* JWT Authentication
* Add Income & Expenses
* Edit Transactions
* Delete Transactions
* Analytics Dashboard
* Pie Chart Visualization
* MongoDB Atlas Integration
* Responsive UI

## 🛠️ Tech Stack

### Frontend

* React.js
* Vite
* Chart.js
* CSS

### Backend

* Node.js
* Express.js
* MongoDB Atlas
* Mongoose
* JWT
* bcryptjs

## 📊 Analytics

* Income Tracking
* Expense Tracking
* Balance Calculation
* Expense Distribution Pie Chart
