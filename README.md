# 💰 Expense Tracker Pro

A full-stack Expense Tracker application built using the MERN Stack (MongoDB, Express.js, React.js, and Node.js).

## 🚀 Live Demo

**Frontend:** https://self-tracker-xi.vercel.app

**Backend API:** https://self-tracker-1mv0.onrender.com

---

## ✨ Features

* User Registration & Login
* JWT Authentication
* Add Income & Expenses
* Update Transactions
* Delete Transactions
* Analytics Dashboard
* Pie Chart Visualization
* MongoDB Atlas Integration
* Responsive UI
* Cloud Deployment

---

## 🛠️ Tech Stack

### Frontend

* React.js
* Vite
* Chart.js
* CSS

### Backend

* Node.js
* Express.js
* MongoDB Atlas
* Mongoose
* JWT Authentication
* bcryptjs

---

## 📊 Analytics Features

* Expense Tracking
